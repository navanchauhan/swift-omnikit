OmniKit is my highly opinionated ultimate personal Swift Library. Swift 6.0+ with strict concurrency.

## OmniUI

Drop in SwiftUI replacement bringing my SwiftUI apps to other platforms.

### OmniUICore

1:1 SwiftUI APIs reimplemented

### OmniUINcursesRenderer

TUI renderer built on top of Ncurses. 
* Scrolling
* Mouse Clicks
* Image support (on supported terminals)
* Animations
* Colour Support

### OmniUIAdwaitaRenderer

GTK Renderer using Adwaita theme pack

### OmniUIWeb

Render to web with opionated styling so you can bring your SwiftUI apps to the web while still looking pretty jazzy.

## OmniAI

### OmniAILLMClient

Unified LLM client for LLM providers with provider preferred APIs being used.
* OpenAI - Responses API
* Anthropic - Messages API
* Gemini - Interactions API

### OmniAIAgent

Agentic loop runtime with tool support and state management

### OmniAICode

Interactive coding agent CLI
